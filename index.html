Who I am

Greetings, I am Riley Halloran, a graduate student at IU and an independent game developer. I specialize as a gameplay programmer with specialties in Artificial Intelligence and Network Science. Currently working for the Cyberinfrastructure for Network Science Center at IU as a Unity developer.

My Work

Erin and the Otherworld

This is a game I am currently working on with a team of fellow students. An unreleased Celtic themed, constellation drawing, rouge-lite game set to release in Q1 2023 where the player as Erin set about sealing the Otherworld to protect their home.

As one of two initial programmers for the game my work has been focused on implementing features for the game like a state machine, constellation drawing elements, and the project structure of the game. I work in tandem with the team to communicate tasks between the programmers and designers. During development I have maintained code-writing-conventions in addition to writing documentation. Examples of my work can be seen following the title link.

Madness Hunters

This is a cancelled project that me and a team of 4 were developing for 5 months. The game is a single player "yank-and-slash" rogue like where the player uses a grappling hook to pull around friendly companions in order to defeat an eldritch threat. My position for this game was the head programmer where I produced all the code and gameplay elements in the project.
